# 0004 Demo 音频素材

本目录内的音频文件为**合成生成**（synthesized）素材，用于固件音频播放链路的回归验证与示例演示。

## 文件位置

- Demo playlist：`assets/demo-playlist/`

## 素材约束（已冻结）

- 单声道（mono）
- 采样率：`8000 Hz`
- 段间静音：由固件插入 `1s`
- 电平：峰值 `<= -6 dBFS`（不削波）
- 覆盖内容：`WAV(PCM16LE)`、`WAV(IMA-ADPCM)`、旋律、扫频（按 `01_`…`06_` 顺序）
- 时长：每段约 `10s`（不强制整秒；具体以各文件样本数为准）

## 预览说明（macOS 常见情况）

- `02_*` / `03_*` / `06_*` 为 `WAV(IMA-ADPCM)`：很多系统自带预览/播放器不支持，出现“无法预览/无法播放”属于正常。
- `01_*` / `04_*` / `05_*` 为 `WAV(PCM16LE)`：应可被大多数播放器直接播放；macOS 可用 `afplay <file.wav>` 验证。
