# 项目硬件选型（mains-aegis）

本文件是整个项目的硬件选型“总纲”，用于沉淀：

- 系统级硬件架构与分块
- 各分块关键器件的“已选/候选/待定”状态与依据
- 需要进一步补齐的输入（把选型从“建议”落到“可下单 BOM”）

## 1. 现状与范围

当前仓库已收录并转换（`mineru`）的数据手册资料包含：

- BMS / 电池包管理：`BQ40Z50-R2`
- 二次保护：`CLM1612P1412`、`SFJ-1412`
- 二级过压保护（OVP）：`BQ296100DSGR`
- 主控 MCU：`ESP32-S3-FH4R2`（ESP32-S3 series；datasheet：`docs/datasheets/esp32-s3-fh4r2/`；配套资料见 `docs/manuals/esp32-s3-*`）

因此本阶段能“确定”的选型可覆盖：电池包管理（含电量计/电量计功能）、保护/均衡与二次保护；其他分块（若项目还有充电/电源/通信/控制等）需要你补充需求与约束后才能继续落地到具体料号。

### 2.5 均衡（cell balancing）

| 子项 | 选型状态 | 说明 |
|---|---|---|
| 均衡策略 | 已定（外部被动均衡） | 采用 TI app note 的外部均衡 **PMOS** 拓扑（BQ40Z50 内部均衡开关 + 外部 PMOS + `Rext`）；目标均衡电流：`200 mA`（平均值） |
| 外部均衡 MOSFET（PMOS） | `UMW SI2305A` | `SOT-23`，`VDS=-20V`，`ID=-4.2A`，`VGS(th) max=-0.5V`，`RDS(on) max 40mΩ@-4.5V / 60mΩ@-2.5V`；资料：`docs/datasheets/UMW_SI2305A/` |
| 外部均衡电阻（Rext） | 已定（200 mA） | `Rext=16Ω`（以 `Vcell≈4.2V` 估算约 `200 mA` 平均均衡电流）；建议 `2512`、`≥2W`、`1%`，并按 PCB 热能力校核 |
| VC 采样滤波（Rvc/Cvc） | 已定（按 BQ40Z50 推荐） | `Rvc=100Ω`（每个 VCx 串联电阻）；`Cvc=0.1µF`（按 datasheet 连接）；建议 `Cvc` 选 `X7R`、`≥10V` 并贴近 IC 放置 |

## 4. 需要你补齐的项目输入（我才能继续把“待定”变成具体料号）

请你按实际项目回答（越具体越好）：

1. 电池包：已确定为 `4S1P`（电芯：EVE 亿纬 `EVE-INR21700/50E`，见 2.0）；已查到电芯满充/截止放电电压（每节 `4.2 V / 2.5 V`，整包 `16.8 V / 10.0 V`）；已知系统连续放电约 `12 A`（DCDC 输出 `19 V / 6.32 A`）；待确认：系统级峰值放电电流（需结合 DCDC 输入限流/启动瞬态）、以及充电电流目标（A）。
2. 保护策略：希望的 OC1/OC2/SHORT 门限电流（A）与延时（ms/us）？
3. 均衡策略：目标均衡电流（mA）与允许的发热/散热方式（贴片电阻尺寸、铜皮、风道）？
4. 温度：充电/放电的温度阈值点（℃）、NTC 的安装位置（贴电芯/贴板/外置探头）？
5. 机械/连接：需要的连接器类型（焊线/插拔/螺栓端子）、允许的板尺寸与层数/铜厚？
