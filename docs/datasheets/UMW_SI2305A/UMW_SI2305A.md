# UMW SI2305A — key parameters (project use: external balancing PMOS)

Package: `SOT-23`  
Type: `P-Channel MOSFET`  
Project use: external cell balancing switch for `BQ40Z50-R2` (TI external PMOS topology)

## Key specs (TA = 25°C)

- `VDS = -20 V`
- `VGS = ±12 V`
- `ID = -4.2 A` (continuous)
- `IDM = -10 A` (pulse)
- `PD = 1.38 W` @ TA=25°C (`0.8 W` @ TA=70°C)
- `VGS(th) max = -0.5 V` (at `ID = -250 µA`)
- `RDS(on) max = 40 mΩ` @ `VGS=-4.5 V`, `ID=-4.2 A`
- `RDS(on) max = 60 mΩ` @ `VGS=-2.5 V`, `ID=-3.4 A`
