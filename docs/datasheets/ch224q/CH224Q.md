# CH224Q — USB‑C PD 受电触发（摘要 + 关键表）

来源：WCH《CH224 手册1》，版本 `2.1`（见 `docs/datasheets/CH224Q/README.md`）。

## 1. 概述（节选）

CH224Q/CH224A 为支持 USB PD3.2 的受电芯片，支持 `PD3.2 EPR / AVS / PPS / SPR` 等，并支持 `BC1.2` 等升压快充协议；支持单电阻配置、I/O 电平配置与 I2C 配置。通过 I2C 可读取协议握手状态与当前 PD 档位额定电流。芯片内置高压 LDO，并集成输出电压检测与过压保护（OVP）。

## 2. 关键特性（节选）

- 输入电压：`4V~30V`
- 支持：`PD3.2 EPR / AVS / PPS / SPR`、`BC1.2` 等
- I2C：`400kHz`，地址 `0x22` 或 `0x23`（7bit，不含读写位）
- 封装：`DFN10 2×2mm`（CH224Q）

## 3. CH224Q 引脚（节选）

| 引脚 | 名称 | 类型 | 说明 |
|---:|---|---|---|
| 0 | GND | P | 散热底板/地 |
| 1 | VHV | P | 工作电源输入（外接 `1µF` 对地电容，注意耐压） |
| 2 | CFG2 / SCL | I, PU | 档位配置输入 2 或 I2C SCL |
| 3 | CFG3 / SDA | I/O, PU | 档位配置输入 3 或 I2C SDA |
| 4 | DP | I/O | USB 总线 |
| 5 | DM | I/O | USB 总线 |
| 6 | CC2 | I/O | Type‑C CC2 |
| 7 | CC1 | I/O | Type‑C CC1 |
| 8 | VBUS | I | 电压检测输入，**需要短接至 VHV** |
| 9 | CFG1 | I | 档位配置输入 1 |
| 10 | PG | OD | Power Good 指示（低有效，可定制） |

## 4. 电压请求配置

### 4.1 单电阻配置（CFG1‑GND）

| 配置电阻 | 请求电压 |
|---:|---:|
| `6.8kΩ` | `9V` |
| `24kΩ` | `12V` |
| `56kΩ` | `15V` |
| `120kΩ` | `20V` |
| `210kΩ` | `28V` |

> 单电阻配置时，`CFG2/CFG3` 可悬空。

### 4.2 I/O 电平配置

| CFG1 | CFG2 | CFG3 | 请求电压 |
|---:|---:|---:|---:|
| 0 | 0 | 0 | 9V |
| 0 | 0 | 1 | 12V |
| 0 | 1 | 1 | 20V |
| 0 | 1 | 0 | 28V |
| 1 | X | X | 5V |

说明：

- “0”表示低电平，外部短接到 GND。
- “1”表示高电平；CFG2/CFG3 内置上拉电阻，支持 `3.3V/5V` 电平输入。

## 5. I2C 寄存器（节选）

### 5.1 `0x09` I2C 状态寄存器（只读）

位定义（BIT=1 表示对应协议握手成功）：

- BIT0：BC 激活
- BIT1：QC2 激活
- BIT2：QC3 激活
- BIT3：PD 激活
- BIT4：EPR 激活

### 5.2 `0x0A` 电压控制寄存器（只写）

请求电压值：

- `0`: 5V
- `1`: 9V
- `2`: 12V
- `3`: 15V
- `4`: 20V
- `5`: 28V
- `6`: PPS 模式
- `7`: AVS 模式

### 5.3 `0x50` 电流数据寄存器（只读）

当前 PD 档位最大可用电流，单位 `50mA`（仅 PD 握手有效）。
